# Architecture Decision Records

We record significant architecture decisions as ADRs.
AI agents: read relevant ADRs before modifying architectural patterns.

## Creating a New ADR

1. Copy `template.md` to a new file: `NNNN-short-title.md` (use the next available number).
2. Fill in each section.
3. Set the status to `Proposed` until the team agrees, then update to `Accepted`.
